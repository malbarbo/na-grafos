Para executar os programas, entre no diretório que contém este arquivo

Para compilar o programa erdos.c

cc -o erdos erdos.c

Para executar os testes

./erdos

Para executar o programa com um arquivo de entrada (teste com outro nome inicial além de erdos!)

./erdos pares.txt erdos

Para executar os testes do programa erdos.py

python3 erdos.py

Para executar o programa com um arquivo de entrada

python3 erdos.py pares.txt erdos
